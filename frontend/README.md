# Revora Frontend

Next.js 14 App Router + TypeScript + React Query. Реализованы вход и обновление сессии, ролевая навигация, фильтры периода/филиала, Dashboard, финансы, продажи, врачи, маркетинг, отчёты, импорт и администрирование.

Локально без Docker backend ожидается на `http://127.0.0.1:8000/api/v1`. Адрес можно изменить через `NEXT_PUBLIC_API_URL`.

```powershell
npm install
npm run dev
```

Production-проверка: `npm run build`.
