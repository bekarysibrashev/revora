# Документы проектирования Revora

- `00_master_kontekst_revora.md` - сводный контекст проекта, decision log.
- `01_analiz_revora.md`, `02_arhitektura_revora.md`, `03_baza_dannyh_revora.md`,
  `04_api_revora.md` - **отсутствуют в этой сессии.** Они упоминаются в мастер-контексте
  как уже пройденные Этапы 1-4, но сами файлы не были загружены/пересозданы в этом
  разговоре. Не воссоздавал их по памяти, чтобы не подменить реальные решения
  выдумкой - при следующей сессии их нужно приложить, иначе Этап 7 (шаги "конфиг",
  "БД", "API") будет идти по краткой сводке из разделов 5-7 `00_master_kontekst_revora.md`,
  а не по полным документам.
- `05_frontend_revora.md` - Этап 5, включая решения №9 (AI-чат = заглушка) и №10 (экспорт).
- `06_ai_moduli_revora.md` - Этап 6, включая решение №11 (аналитика = фоновые джобы).
